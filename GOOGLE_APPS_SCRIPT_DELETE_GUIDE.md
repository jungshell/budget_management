# Google Apps Script 프로젝트 삭제 가이드

## 방법 1: 프로젝트 설정에서 삭제 (권장)

### 단계별 가이드

1. **Google Apps Script 편집기 열기**
   - https://script.google.com 접속
   - 삭제할 프로젝트 선택

2. **프로젝트 설정 열기**
   - 좌측 상단의 **"프로젝트 설정"** 아이콘 클릭 (톱니바퀴 아이콘 ⚙️)
   - 또는 상단 메뉴에서 **"프로젝트 설정"** 클릭

3. **프로젝트 삭제**
   - 설정 화면 하단으로 스크롤
   - **"프로젝트 삭제"** 버튼 클릭
   - 확인 대화상자에서 **"삭제"** 클릭

---

## 방법 2: 프로젝트 목록에서 삭제

### 단계별 가이드

1. **Google Apps Script 홈 화면으로 이동**
   - https://script.google.com 접속
   - 또는 편집기에서 좌측 상단의 **"Apps Script"** 로고 클릭

2. **프로젝트 찾기**
   - 프로젝트 목록에서 삭제할 프로젝트 찾기
   - "budget management" 또는 "예산관리시스템 연동" 등

3. **프로젝트 삭제**
   - 프로젝트 위에 마우스 오버
   - 나타나는 **"삭제"** 아이콘 (휴지통 아이콘 🗑️) 클릭
   - 또는 프로젝트를 우클릭 > **"삭제"** 선택
   - 확인 대화상자에서 **"삭제"** 클릭

---

## 방법 3: 기존 배포만 삭제하고 새로 시작

기존 프로젝트는 유지하고 배포만 새로 하고 싶은 경우:

1. **배포 관리 열기**
   - Google Apps Script 편집기에서
   - **"배포"** > **"배포 관리"** 클릭

2. **기존 배포 삭제**
   - "활성" 섹션에서 기존 배포 찾기
   - 배포 옆의 **"삭제"** 버튼 (휴지통 아이콘) 클릭
   - 확인

3. **새 배포 생성**
   - **"새 배포"** 클릭
   - 유형: "웹 앱"
   - 설정 입력 후 배포

---

## ⚠️ 주의사항

### 삭제 전 확인사항

1. **Web App URL 백업**
   - 삭제하기 전에 Web App URL을 복사해두세요
   - 나중에 필요할 수 있습니다

2. **코드 백업**
   - 코드를 복사해두세요
   - 또는 `GOOGLE_APPS_SCRIPT_COMPLETE_CODE.txt` 파일이 있으므로 안전합니다

3. **배포 정보 확인**
   - 현재 배포가 사용 중인지 확인
   - 삭제하면 Web App이 작동하지 않습니다

---

## ✅ 삭제 후 다음 단계

### 새 프로젝트 생성

1. **"새 프로젝트"** 클릭
2. 프로젝트 이름: **"예산관리시스템 연동"**
3. **코드 복사**
   - `GOOGLE_APPS_SCRIPT_COMPLETE_CODE.txt` 파일 열기
   - 전체 내용 복사 (Ctrl+A, Ctrl+C)
4. **붙여넣기**
   - Google Apps Script 편집기에 붙여넣기 (Ctrl+V)
   - 저장 (Ctrl+S)
5. **새로 배포**
   - "배포" > "새 배포"
   - 설정 입력 후 배포
   - 새 Web App URL 복사

### 설정 업데이트

1. 예산 관리 시스템 설정 페이지로 이동
2. 새 Web App URL 입력
3. Google Sheets ID 비우기
4. 저장

---

## 🔍 문제 해결

### 삭제 버튼이 보이지 않는 경우

1. **프로젝트 설정 확인**
   - 프로젝트 설정 화면에서 하단으로 스크롤
   - "프로젝트 삭제" 버튼이 있을 수 있습니다

2. **권한 확인**
   - 프로젝트 소유자인지 확인
   - 공유된 프로젝트는 삭제할 수 없을 수 있습니다

3. **브라우저 새로고침**
   - 페이지를 새로고침 (F5)
   - 다시 시도

가장 빠른 방법:

3. **삭제 아이콘** 클릭
