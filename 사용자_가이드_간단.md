# 📱 예산 관리 시스템 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 1️⃣ 로그인
```
🔐 Google 계정 또는 이메일/비밀번호로 로그인
```

### 2️⃣ 예산 추가
```
➕ "예산 추가" 버튼 클릭
📝 사업명, 부서명, 예산 금액 입력
💾 "저장" 클릭
```

### 3️⃣ AI 분석
```
🤖 "예산 분석 (AI)" 메뉴 클릭
💬 질문 입력 (예: "도비 상위 5개 사업은?")
✨ 결과 확인
```

---

## 📊 핵심 기능

### 예산 관리
```
➕ 추가  →  ✏️ 수정  →  🗑️ 삭제  →  🔍 검색
```

### 파일 업로드
```
📁 Excel/CSV 파일 선택  →  자동 매핑  →  업로드 완료
```

### 데이터 내보내기
```
📥 내보내기 버튼  →  형식 선택 (Excel/CSV/PDF)  →  다운로드
```

### AI 기능
```
🤖 자연어 질문  →  📄 자동 리포트  →  🏷️ 자동 분류  →  💡 예산 제안
```

---

## 🎯 주요 메뉴

| 메뉴 | 아이콘 | 설명 |
|------|--------|------|
| 대시보드 | 🏠 | 예산 요약 및 차트 |
| 예산 관리 | 📊 | 예산 CRUD 작업 |
| 파일 업로드 | 📁 | Excel/CSV 업로드 |
| 예산 분석 (AI) | 🤖 | AI 질의응답 |
| 리포트 | 📄 | 리포트 생성/공유 |
| 설정 | ⚙️ | 시스템 설정 |

## 💡 자주 사용하는 기능

### 🔍 빠른 검색
```
검색창에 키워드 입력 → 실시간 필터링
```

### 📊 컬럼 필터링
```
컬럼 헤더 클릭 → 해당 재원이 있는 항목만 표시
```

### 🎤 음성 입력
```
마이크 아이콘 클릭 → 음성으로 입력
```

### 🤖 AI 자동 분류
```
예산 추가 시 "AI 자동 분류" 버튼 → 카테고리/태그 자동 제안
```

## ⚙️ 필수 설정

### 1. Groq API 키 (AI 기능 사용 시)
```
1. https://console.groq.com 접속
2. API 키 발급
3. 설정 → AI 설정에서 입력
4. API 테스트로 확인
```

### 2. Google Sheets 연동 (선택)
```
1. Google Apps Script URL 입력
2. Google Sheets ID 입력
3. 연동 테스트
```

## ⌨️ 키보드 단축키

| 단축키 | 기능 |
|--------|------|

## 🆘 문제 해결

### ❌ 로그인 안 됨
→ 관리자에게 권한 요청

### ❌ 저장 안 됨
→ 필수 항목 확인, 숫자 형식 확인

### ❌ AI 작동 안 함
→ API 키 설정 확인

### ❌ 파일 업로드 안 됨
→ 파일 형식 확인 (.xlsx, .csv)
