## 🚀 빠른 시작 가이드

### 1. 실제 사용 테스트 시작하기

1. **예산 데이터 입력**
   - http://localhost:3000/budgets 접속
   - 실제 예산 데이터 입력

2. **AI 분석 테스트**
   - http://localhost:3000/analysis 접속
   - "총 예산은 얼마인가요?" 같은 질문 입력
   - 결과 확인

3. **Google Apps Script 연동 테스트**
   - 설정 페이지에서 Google Apps Script URL 입력
   - 예산 페이지에서 "Google Sheets 동기화" 버튼 클릭
   - Google Sheets에서 데이터 확인

4. **백업 테스트**
   - 설정 페이지에서 백업 생성
   - 데이터 수정
   - 백업에서 복원

### 2. 성능 확인

```bash
# 개발 서버 실행 중
# 브라우저에서 http://localhost:3000 접속
# 개발자 도구 (F12) > Performance 탭에서 성능 측정
```

### 3. 모바일 테스트

1. **모바일 브라우저에서 접속**
   - 같은 네트워크에서 `http://[컴퓨터IP]:3000` 접속
   - 또는 ngrok 등으로 외부 접근 가능하게 설정

2. **OCR 기능 테스트**
   - 모바일에서 카메라로 문서 촬영
   - OCR로 텍스트 추출 확인
