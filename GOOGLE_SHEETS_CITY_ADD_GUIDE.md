# Google Sheets 시군 추가 가이드

## 시군을 쉽게 추가하는 방법

### 방법 1: Google Sheets에서 직접 추가 (가장 쉬운 방법)

1. **"시군비 상세" 탭 열기**
   - Google Sheets 하단의 "시군비 상세" 탭 클릭

2. **새 시군 컬럼 추가**
   - 마지막 컬럼 오른쪽에 마우스 오버
   - 우클릭 > **"열 삽입"** 선택
   - 또는 마지막 컬럼 오른쪽 빈 셀에 직접 입력

3. **헤더 입력**
   - 첫 번째 행에 `출연금-시군명` 입력 (예: `출연금-공주시`)
   - 그 다음 행에 `보조금-시군명` 입력 (예: `보조금-공주시`)

4. **데이터 입력**
   - 각 사업별로 출연금과 보조금 입력
   - 숫자 형식은 자동으로 천 단위 구분 쉼표가 적용됩니다

5. **메인 시트에도 반영**
   - 메인 시트(회차시트)로 이동
   - 마지막 컬럼 오른쪽에 새 시군 컬럼 추가
   - `출연금-시군명`, `보조금-시군명` 헤더 입력
   - 데이터 입력

---

### 방법 2: 예산 관리 시스템에서 추가

1. **예산 관리 페이지로 이동**
   - 예산 관리 시스템 열기
   - 수정할 예산 항목 선택

2. **시군비 상세 입력**
   - 예산 수정 다이얼로그 열기
   - "출연금" 또는 "보조금" 섹션에서 "시군비" 클릭
   - 새 시군 추가 버튼 클릭
   - 시군명과 금액 입력
   - 저장

3. **Google Sheets로 내보내기**
   - "Google Sheets로 내보내기" 클릭
   - 새 시군이 자동으로 추가됩니다

## 팁

### 빠른 복사 방법

1. **기존 시군 컬럼 복사**
   - 기존 시군 컬럼 선택 (예: 출연금-천안시, 보조금-천안시)
   - Ctrl+C (복사)

2. **새 위치에 붙여넣기**
   - 마지막 컬럼 오른쪽에 붙여넣기 (Ctrl+V)
   - 헤더만 수정 (시군명 변경)

3. **데이터만 수정**
   - 각 행의 금액만 수정하면 됩니다

---

## 주의사항

- 시군명은 정확하게 입력하세요 (예: "천안시" O, "천안" X)
- 출연금과 보조금은 각각 별도 컬럼으로 입력하세요
- 숫자는 천 단위 구분 쉼표가 자동으로 적용됩니다
