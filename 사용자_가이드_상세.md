## 📊 기본 기능

### 1. 예산 관리

#### ➕ 예산 항목 추가

```
1. "예산 추가" 버튼 클릭 ➕
2. 필수 정보 입력:
   ├─ 사업명: 예산 항목 이름
   ├─ 소관부서: 담당 부서
   └─ 예산 금액 입력
3. 재원별 예산 입력:
   ├─ 출연금: 도비, 시군비
   ├─ 보조금: 국비, 도비, 시군비
   └─ 자체재원
4. "저장" 버튼 클릭 💾
```

**💡 팁:**
- 음성 입력 🎤 버튼으로 사업명/부서명 빠르게 입력
- AI 자동 분류 🤖 버튼으로 카테고리 자동 제안

#### ✏️ 예산 항목 수정

```
1. 수정할 항목의 "수정" 아이콘 클릭 ✏️
2. 정보 수정
3. 변경 이유 입력 (선택사항)
4. "저장" 버튼 클릭 💾
```

#### 🗑️ 예산 항목 삭제

```
1. 삭제할 항목의 "삭제" 아이콘 클릭 🗑️
2. 확인 대화상자에서 "확인" 클릭
```

#### 🔍 검색 및 필터링

**검색:**
- 검색창에 사업명 또는 부서명 입력
- 실시간으로 필터링됨

**필터링:**
- 부서별 필터: 드롭다운에서 부서 선택
- 재원별 필터: 컬럼 헤더 클릭으로 해당 재원이 있는 항목만 표시

**정렬:**
- 컬럼 헤더 클릭으로 정렬
- "기본정렬" 버튼으로 초기 상태로 복원

#### 📊 컬럼 설정

```
1. "컬럼 설정" 아이콘 클릭 👁️
2. 표시/숨김 토글
3. 컬럼 너비 조정
4. 컬럼 순서 변경 (드래그 앤 드롭)
```

### 2. 파일 업로드

#### 📁 Excel/CSV 파일 업로드

```
1. "파일 업로드" 메뉴 클릭
2. 파일 선택 버튼 클릭
3. Excel (.xlsx, .xls) 또는 CSV 파일 선택
4. 자동 매핑 확인
5. "업로드" 버튼 클릭
```

**지원 형식:**
- 📊 Excel: `.xlsx`, `.xls`
- 📄 CSV: `.csv`
- 🍎 Numbers: `.numbers`

**자동 인식 항목:**
- 사업명
- 소관부서
- 총 예산액
- 재원별 예산 (국비/도비/시군비/자체)

### 3. 데이터 내보내기

#### 📥 내보내기 형식

```
1. "내보내기" 버튼 클릭 📥
2. 형식 선택:
   ├─ 📊 Excel (.xlsx)
   ├─ 📄 CSV (.csv)
   ├─ 📑 PDF (.pdf)
   ├─ 📋 JSON (.json)
   └─ 📝 HWP (.hwp)
3. 파일 다운로드
```

#### 📤 Google Sheets 동기화

```
1. 설정에서 Google Apps Script URL 입력
2. 예산 관리 페이지에서 "Google Sheets 동기화" 클릭
3. Google Sheets에서 데이터 확인
```

## 🤖 AI 기능

### 1. 자연어 예산 분석

#### 💬 질문하기

```
1. "예산 분석 (AI)" 메뉴 클릭
2. 질문 입력 예시:
   ├─ "올해 본예산에서 도비 상위 5개 사업을 알려줘"
   ├─ "전년 대비 예산 증감률이 가장 큰 사업은?"
   ├─ "국비가 가장 많이 투입된 사업 3개를 분석해줘"
   └─ "시군비 비중이 높은 사업들을 알려줘"
3. "AI 분석 요청" 버튼 클릭
4. 결과 확인
```

**💡 팁:**
- 자연스러운 한국어로 질문 가능
- 예시 질문 클릭으로 빠르게 입력

#### 📄 자동 리포트 생성

```
1. "자동 리포트 생성" 버튼 클릭
2. 리포트 기간 선택:
   ├─ 주간
   ├─ 월간
   ├─ 분기
   └─ 연간
3. "리포트 생성" 버튼 클릭
4. 생성된 리포트 다운로드
```

### 2. AI 자동 분류

#### 🏷️ 카테고리 및 태그 자동 생성

```
1. 예산 추가/수정 다이얼로그 열기
2. 사업명과 소관부서 입력
3. "AI 자동 분류" 버튼 클릭 🤖
4. 제안된 카테고리와 태그 확인
5. 필요시 수정 후 저장
```

**제안 항목:**
- 📂 카테고리: 정보화, 문화, 산업, 교육, 복지, 인프라, 기타
- 🏷️ 태그: 관련 키워드 자동 생성
- 💭 분류 이유: AI가 제안한 이유 설명

### 3. AI 예산 제안

#### 💡 최적 예산 분배 제안

```
1. 대시보드 페이지 이동
2. "AI 기반 예산 제안" 섹션 찾기
3. "AI 제안 생성" 버튼 클릭 🔄
4. 부서별 예산 제안 확인:
   ├─ 현재 예산 vs 제안 예산
   ├─ 차이액
   ├─ 우선순위 (높음/중간/낮음)
   └─ 제안 사유
```

**제안 내용:**
- 부서별 예산 분배 균형
- 재원 유형별 최적 비율
- 효율성 개선 방안

### 4. AI 리스크 분석

#### ⚠️ 예산 리스크 평가

```
1. 대시보드 페이지 이동
2. "AI 기반 리스크 분석" 섹션 찾기
3. "AI 리스크 분석" 버튼 클릭 🔄
4. 리스크 분석 결과 확인:
   ├─ 전체 리스크 점수 (0-100)
   ├─ 리스크 카테고리별 상세 정보
   ├─ 영향받는 항목
   └─ 개선 제안
```

**리스크 레벨:**
- 🔴 **높음**: 즉시 조치 필요
- 🟡 **중간**: 주의 필요
- 🔵 **낮음**: 모니터링 권장

## 🛠️ 문제 해결

### 1. 로그인 문제

**문제:** 로그인이 안 됩니다
```
✅ 해결 방법:
1. 이메일/비밀번호 확인
2. 관리자에게 권한 요청
3. 브라우저 캐시 삭제 후 재시도
```

### 2. 데이터 저장 오류

**문제:** 예산 항목이 저장되지 않습니다
```
✅ 해결 방법:
1. 필수 항목 입력 확인 (사업명, 부서명)
2. 숫자 형식 확인 (쉼표 제거)
3. 총 예산액과 세부 항목 합계 일치 확인
4. 네트워크 연결 확인
```

### 3. AI 기능 오류

**문제:** AI 분석이 작동하지 않습니다
```
✅ 해결 방법:
1. Groq API 키 설정 확인
2. API 키 유효성 확인 (API 테스트 버튼)
3. 네트워크 연결 확인
4. 브라우저 콘솔에서 오류 메시지 확인
```

### 4. 파일 업로드 오류

**문제:** 파일이 업로드되지 않습니다
```
✅ 해결 방법:
1. 파일 형식 확인 (.xlsx, .xls, .csv)
2. 파일 크기 확인 (10MB 이하 권장)
3. 파일이 손상되지 않았는지 확인
4. Excel 파일의 시트 구조 확인
```

### 5. 성능 문제

**문제:** 페이지가 느립니다
```
✅ 해결 방법:
1. 브라우저 캐시 삭제
2. 불필요한 데이터 삭제
3. 필터링으로 표시 항목 수 줄이기
4. 네트워크 연결 확인
```
